## OptiKey 是什么？

OptiKey 是一个开源的眼球跟踪软件。Julius 用三年半的业余时间，为渐冻开发。详情请看伯乐在线的文章《[历时三年半，这位程序员在业余时间为渐冻人做了一个开源的眼球跟踪软件](http://blog.jobbole.com/88485/)》。
